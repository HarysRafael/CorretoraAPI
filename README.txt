Api criada com Flask e PostgreSQL, voltada para o aluguel de imóveis com funcionalidades de Criar, Listar, Atualizar e Deletar. Possui as entidades: Proprietários, Inquilinos, Corretores e Imóveis e Aluguéis, onde é possível listar o total e os detalhes de Aluguéis, a quantida de Aluguéis por corretor, além de Imóveis disponíveis e alugados.
Para execução local, além do Python 3, é necessário a instalação das bibliotecas flask, sqlalchemy, flask-sqlalchemy, psycopg2-binary e flask-migrate.
